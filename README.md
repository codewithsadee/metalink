### Run Locally

To run **Metalink** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/metalink.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/metalink.git
```
